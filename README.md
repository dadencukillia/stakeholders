# Stakeholders 👥

> Platform for managing freelancer projects with progress monitoring and AI analytics

## 📋 About the Project

**Stakeholders** is a web platform that connects freelancers with their clients. The platform provides tools for project management, code sharing, and real-time AI insights.

### Key Features

- 📊 **Project Management** — Each project has its own Git repository
- 🔗 **Link Sharing** — Send clients secure links for access
- ⏱️ **WakaTime Integration** — Track productivity automatically
- 🤖 **AI Analytics** — Intelligent project status summaries
- 👁️ **Real-time Monitoring** — View files and freelancer status
- 🔐 **Access Control** — Enable/disable code access

## 🛠️ Tech Stack

### Backend
- [**Go**](https://go.dev) — primary language
- [**Docker**](https://docker.com) — containerization
- [**SQLc**](https://sqlc.dev) — type-safe SQL queries
- [**Goose**](https://pressly.github.io/goose) — database migrations

### Frontend
- [**Bun**](https://bun.com) — primary bundler runtime
- [**Vite**](https://vite.dev) — type-safe development
- [**Astro**](https://astro.build) — frontend framework
- [**Shadcn**](https://ui.shadcn.com) — components
- [**TailwindCSS**](https://tailwindcss.com) — styling
